# Supermercado Támaisbarato
Trabalho de programação web utilizando HTML/CSS

## 📌 Objetivo do Sistema
Este projeto consiste em um site de supermercado com o objetivo de simular um e-commerce.  
O usuário pode visualizar produtos, navegar entre páginas, utilizar um carrinho de compras e entrar em contato.

---

## 🚀 Tecnologias Utilizadas

- **HTML5**  
  Utilizado para estruturar todas as páginas do site, como cabeçalho, conteúdo principal e rodapé.

- **CSS3**  
  Utilizado para estilização do site, incluindo cores, layout e organização visual dos elementos.

---

## 🗂️ Estrutura do Sistema

O sistema é composto pelas seguintes páginas:

- **index.html**  
  Página principal com produtos em destaque e seção de novidades.

- **produtos.html**  
  Página com listagem de diversos produtos disponíveis.

- **carrinho.html**  
  Página que simula um carrinho de compras, apresentando produtos e valor total.

- **faleconosco.html**  
  Página com formulário para contato do usuário.

---

## ▶️ Instruções para Execução

Para executar o sistema, siga os passos abaixo:

1. Baixe ou salve todos os arquivos do projeto em uma mesma pasta.
2. Certifique-se de que as imagens estão dentro da pasta correta.
3. Abra o arquivo `index.html` em um navegador (Google Chrome, Microsoft Edge, etc.).
4. Utilize o menu de navegação para acessar as demais páginas do sistema.

---

## 📷 Observações

- Este projeto é apenas uma simulação de e-commerce.
- Não possui integração com banco de dados ou sistema de pagamento.
